# findbusstop

> 日本語のREADMEはこちらです: [README.ja.md](README.ja.md)

Find bus stops with 5-star open data.

## Features
- Searches for nearby bus stops using open data
- Displays bus stop information on a map
- Supports multiple languages (English, Japanese, and others)

## Usage
1. Open the web page in a browser.
2. Click the "Nearest" button to find the nearest bus stops.
3. Navigate through the list of bus stops using the "Previous" and "Next" buttons.

## Data / API
This project uses the SPARQL API provided by the Open Data Portal (ODP) to retrieve bus stop information.
