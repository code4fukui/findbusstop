# バス停を探せ 5つ星オープンデータ版

公共交通機関の運行情報をオープンデータから表示するウェブアプリです。Google Mapsと5つ星オープンデータを使って、近隣のバス停を簡単に検索できます。

## デモ
https://codeforfukui.github.io/findbusstop/

## 機能
- 現在地から最寄りのバス停を検索
- バス停の位置情報とルート情報を表示
- 言語切り替え（日本語、英語など）

## 必要環境
- Webブラウザ
- GoogleアカウントでのAPIキー取得

## 使い方
1. GoogleアカウントでAPIキーを取得し、`lib/gmap.js`の`API_KEY`に設定します。
2. `index.html`を開くと、位置情報の許可を求められるので、許可します。
3. 現在地からの最寄りのバス停を表示します。

## データ・API
- 5つ星オープンデータ
- Google Maps API
